Pemesanan Tiket Museum Pusat TNI Angkatan Laut (Muspusal)

Dalam Project Tersebut Saya di tugaskan sebagai Front End Developer, saya bertanggung jawab untuk merancang dan mengembangkan antarmuka pengguna untuk sistem pemesanan tiket di Museum Pusat TNI Angkatan Laut (Muspusal). Proyek ini melibatkan pembuatan halaman web yang responsif, mudah digunakan, dan menarik secara visual untuk memfasilitasi pengalaman pemesanan tiket yang lancar bagi pengunjung.

    Tanggung Jawab dan Kontribusi :
  • Desain Antarmuka Pengguna: Mengembangkan desain yang intuitif dan user-friendly, memastikan navigasi yang mudah dan pengalaman pengguna yang menyenangkan.

  • Pengembangan Front End: Menggunakan teknologi web terbaru dengan bahasa Csharp , HTML, CSS, dan JavaScript untuk membangun halaman web yang responsif dan dinamis.

  • Kolaborasi Tim: Bekerja sama dengan desainer UX/UI dan tim backend untuk mengintegrasikan fungsionalitas dan memastikan kinerja yang optimal.

  • Pengujian dan Debugging: Melakukan pengujian menyeluruh untuk memastikan semua fitur berfungsi dengan baik dan memperbaiki bug yang ditemukan.

  • Optimalisasi Kinerja: Memastikan bahwa halaman web dioptimalkan untuk kecepatan dan efisiensi, termasuk optimasi gambar dan caching.

    Teknologi yang Digunakan :
  • HTML5

  • CSS3

  • JavaScript

  • Framework CSS (misalnya Bootstrap atau Tailwind CSS)

  • Frontend (.NetCore)

  • Kunjungi Website

Pembuatan Aplikasi E-Kantin

Saya mengembangkan aplikasi e-kantin yang berfungsi sebagai platform digital untuk mengelola transaksi di kantin. Aplikasi ini memiliki tiga peran utama: user (pembeli), admin (kasir kantin), dan super admin (Pengelola). Dengan menggunakan bahasa pemrograman Python dan framework Flask, aplikasi ini dirancang untuk memberikan pengalaman yang seamless bagi semua pengguna hingga ke tahap pembayaran yang dilakukan melalui payment gateway Midtrans.

    Tanggung Jawab dan Kontribusi :
  • Perancangan Sistem: Mendesain arsitektur sistem dengan jelas, termasuk peran dan hak akses masing-masing pengguna (user, admin, super admin).

  • Pengembangan Backend: Menggunakan Python dan Flask untuk mengembangkan logika backend yang efisien dan aman.

  • Pengembangan Frontend: Mengintegrasikan antarmuka pengguna yang interaktif dan responsif untuk pengalaman yang optimal.

  • Manajemen Database: Mendesain dan mengelola database untuk menyimpan data transaksi, pengguna, dan inventaris kantin.

  • Integrasi Pembayaran: Mengintegrasikan Midtrans sebagai payment gateway untuk memfasilitasi transaksi yang aman dan cepat.

    Teknologi yang Digunakan :
  • Python

  • Flask

  • HTML5, CSS3, JavaScript

  • Database NoSQL (Google Clouds)

  • Payment Gateway Midtrans
